## Features

- Built on [LazyVim](https://github.com/LazyVim/LazyVim)
- Custom keybindings for efficient navigation and editing
- Optimized plugin management and performance
- Integrated tools for modern development workflows

## Installation

Clone this repository to your Neovim configuration directory:

```bash
git clone https://github.com/sifenfisaha/nvim.git ~/.config/nvim
```

Then open Neovim to automatically install and configure plugins:

```bash
nvim
```
